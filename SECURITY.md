# Security Policy

Agent Tools Hub lists third-party tools. Inclusion does not mean the project has audited or endorsed the listed repository.

## Reporting A Vulnerability

If you find a security issue in this site, please open a private security advisory if the repository supports it, or contact the repository owner.

If the issue is in a listed third-party tool, report it to that tool's maintainers first.

## Data Safety

Tool entries should only include public metadata:

- Public GitHub URL
- Repository owner and name
- Public description
- Public tags and classification

Do not add secrets, private endpoints, private tokens, or non-public contact information to `data/skills.json`.
